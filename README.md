# Research-Agent
AI research agent that turns any topic into a structured, source backed course PDF powered by Claude + live web search.
Nexus is an AI powered research agent that takes any topic, searches the web in real time, and generates a professionally formatted multi lesson course PDF. Powered by Anthropic's Claude Sonnet, it structures content into progressive lessons each with subheadings, key takeaways, and real cited sources then exports everything as a polished PDF. Comes with both a CLI and a minimal web UI.
